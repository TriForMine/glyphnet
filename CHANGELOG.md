# Changelog

## 0.1.0

- Initial workspace scaffold.
- Deterministic binary frame format.
- Non-square `RibbonWeave` reference layout with continuous strokes, side
  totems, chevron rails, and a center phase trace.
- Ribbon renderer defaults and fixed signature rails to avoid QR/Data Matrix/DotCode confusion.
- Static encode-render-decode path.
- Burst frame encoding and assembly.
- CLI, WASM bridge, browser/mobile SDK scaffolds.
- Unit tests, property tests, fuzz targets, benchmarks, and CI workflows.
